## list

```bash
fp-appimage-updater list
```

Ostendit omnes applicationes configuratas una cum versione earum localiter registrata et statu integrationis currenti (praesentia symlink, inscriptio desktopi et icon).

## remove

```bash
fp-appimage-updater remove <nomen-app>
fp-appimage-updater remove -a
```

Removet binarium, symlink, icones extractos et fasciculos desktopi applicationis. Hoc non delet fasciculum formulae - applicatio reinstallari potest exsequendo `update` iterum.

### Vexilla

<ParamField query="-a, --all" type="boolean">
  Removet omnes applicationes installatas simul.
</ParamField>

### Exempla

<CodeGroup>

```bash Unica applicatio
# Removere unicam applicationem
fp-appimage-updater remove whatpulse
```

```bash Omnes applicationes
# Removere omnes applicationes installatas
fp-appimage-updater remove -a
```

</CodeGroup>

<Warning>
  `remove -a` removet omnes AppImages installatos, eorum symlinks, icones et inscriptiones desktopi in unico gradu. Hoc reverti non potest sine exsecutione `update` iterum.
</Warning>

## self-update

```bash
fp-appimage-updater self-update
```

Petit emissionem recentissimam ipsius `fp-appimage-updater` et binarium currentem substituit.

### Vexilla

<ParamField query="--pre-release" type="boolean">
  Includit versiones prae-emissas cum verificando si binarium recentius adest.
</ParamField>

```bash
# Renovare ad emissionem stabilem recentissimam
fp-appimage-updater self-update

# Renovare includens versiones prae-emissas
fp-appimage-updater self-update --pre-release
```

## completion

```bash
fp-appimage-updater completion <shell>
```

Generat scriptum completionis shell pro shell specificato et scribit id in exitum normalem. Shell sustenta: `bash`, `zsh`, `fish`.

### Exempla

<CodeGroup>

```bash Bash
# Habilitare pro sessione bash currenti
source <(fp-appimage-updater completion bash)

# Habilitare permanenter pro bash
fp-appimage-updater completion bash >> ~/.bashrc
```

```bash Zsh
# Habilitare pro sessione zsh currenti
source <(fp-appimage-updater completion zsh)

# Habilitare permanenter pro zsh
fp-appimage-updater completion zsh >> ~/.zshrc
```

```bash Fish
# Habilitare permanenter pro fish
fp-appimage-updater completion fish > ~/.config/fish/completions/fp-appimage-updater.fish
```

</CodeGroup>

<Tip>
  Recarica shell tuum vel exsequere `source ~/.bashrc` (vel aequivalens) ut completiones effectum habeant.
</Tip>
