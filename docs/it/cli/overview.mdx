Invoca `fp-appimage-updater` cum conspectu optionali vexillorum globalium ante submandatum:

```bash
fp-appimage-updater [OPTIONS] <COMMAND>
```

## Vexilla Globalia

Haec vexilla applicantur omni mandato et ante submandatum collocanda sunt.

<ParamField query="-c, --config <PATH>" type="string">
  Superat directorium configurationis defaltum. Utere hoc ad indicandum locum configurationis personalis loco `~/.config/fp-appimage-updater/`.
</ParamField>

<ParamField query="--json" type="boolean">
  Emittit exitum JSON legibilem a machina loco tabularum et linearum status legibilium ab hominibus. Operatur cum `init`, `validate`, `doctor`, `list`, `check`, `update` et `remove`.
</ParamField>

<ParamField query="-h, --help" type="boolean">
  Ostendit informationes auxilii et exit.
</ParamField>

<ParamField query="-V, --version" type="boolean">
  Ostendit versionem currentem et exit.
</ParamField>

## Gruppos de Mandatos

### Configuration e Diagnostica

Iste mandatos te adjuta a comenciar e assecura que tu ambiente local es configurate correctemente.

| Mandato | Description |
|---|---|
| [`init`](/la/cli/init) | Initialisa fasciculos configurationis initiales |
| [`validate`](/la/cli/check-update#validate) | Validat fasciculos formulae applicationis |
| [`doctor`](/la/cli/check-update#doctor) | Exsequitur verificatiónes sanitatis pro configuratione locali |

### Gestion de Applicationes

Le mandatos principal pro mantener tu AppImages actualisate e gerer lor stato de installation.

| Mandato | Description |
|---|---|
| [`check`](/la/cli/check-update#check) | Verifica si hay actualisationes disponibile |
| [`update`](/la/cli/check-update#update) | Actualisa le applicationes a lor versiones le plus recente |
| [`list`](/la/cli/manage#list) | Enumerat omnes applicationes configuratas et earum statum |
| [`remove`](/la/cli/manage#remove) | Removet applicationem et eius integrationes |

### Utilitates de Systema

Mandatos pro mantener le actualisator mesme e meliorar tu experientia in le terminal.

| Mandato | Description |
|---|---|
| [`self-update`](/la/cli/manage#self-update) | Renovat ipsum fp-appimage-updater |
| [`completion`](/la/cli/manage#completion) | Generat scripta completionis shell |

<Note>
  Transmitte `--json` ante quodlibet mandatum sequens ad obtinendum exitum JSON structuratum: `init`, `validate`, `doctor`, `list`, `check`, `update`, `remove`.
</Note>

## Exempla

<CodeGroup>

```bash Exitus JSON
# Enumera omnes applicationes in formato JSON
fp-appimage-updater --json list
```

```bash Config Personalis
# Verifica renovationes utendo directorio configurationis personali
fp-appimage-updater -c ~/config-mea check
```

```bash JSON et jq
# Renovat unicam applicationem et extrahit URL dēscriptionis ex JSON
fp-appimage-updater --json update whatpulse | jq '.[0].download_url'
```

</CodeGroup>
