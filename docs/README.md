# fp-appimage-updater Docs

This directory contains the Mintlify docs for fp-appimage-updater.

## What to edit

- `docs.json` controls navigation and site metadata.
- MDX files under `docs/` define the published pages.
- Keep the docs aligned with the command-line behavior and the YAML recipe format.

## Style

- Use active voice and second person.
- Keep sentences short and direct.
- Use sentence case for headings.
- Format commands, paths, file names, and code references with backticks.

## Local preview

If you want to preview the docs locally, use Mintlify's local dev workflow from the docs site instructions.
