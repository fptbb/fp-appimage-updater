Invoque `fp-appimage-updater` com um conjunto opcional de flags globais seguido de um subcomando:

```bash
fp-appimage-updater [OPTIONS] <COMMAND>
```

## Flags globais

Essas flags se aplicam a todo comando e devem ser colocadas antes do subcomando.

<ParamField query="-c, --config <PATH>" type="string">
  Sobrescreve o diretório de configuração padrão. Use isso para apontar para um local de configuração personalizado em vez de `~/.config/fp-appimage-updater/`.
</ParamField>

<ParamField query="--json" type="boolean">
  Emite saída JSON legível por máquina em vez de tabelas e linhas de status legíveis por humanos. Funciona com `init`, `validate`, `doctor`, `list`, `check`, `update` e `remove`.
</ParamField>

<ParamField query="-h, --help" type="boolean">
  Exibe informações de ajuda e sai.
</ParamField>

<ParamField query="-V, --version" type="boolean">
  Exibe a versão atual e sai.
</ParamField>

## Grupos de Comandos

### Configuração e Diagnóstico

Estes comandos ajudam você a começar e garantem que seu ambiente local esteja configurado corretamente.

| Comando | Descrição |
|---|---|
| [`init`](/pt-BR/cli/init) | Inicializa arquivos de configuração iniciais |
| [`validate`](/pt-BR/cli/check-update#validate) | Valida arquivos de receita de aplicativo |
| [`doctor`](/pt-BR/cli/check-update#doctor) | Executa verificações de saúde para a configuração local |

### Gerenciamento de Aplicativos

Os comandos principais para manter seus AppImages atualizados e gerenciar seu estado de instalação.

| Comando | Descrição |
|---|---|
| [`check`](/pt-BR/cli/check-update#check) | Verifica se há atualizações disponíveis |
| [`update`](/pt-BR/cli/check-update#update) | Atualiza os aplicativos para suas versões mais recentes |
| [`list`](/pt-BR/cli/manage#list) | Lista todos os aplicativos configurados e seu status |
| [`remove`](/pt-BR/cli/manage#remove) | Remove um aplicativo e suas integrações |

### Utilitários do Sistema

Comandos para manter o próprio atualizador e melhorar sua experiência no terminal.

| Comando | Descrição |
|---|---|
| [`self-update`](/pt-BR/cli/manage#self-update) | Atualiza o próprio fp-appimage-updater |
| [`completion`](/pt-BR/cli/manage#completion) | Gera scripts de conclusão de shell |

<Note>
  Passe `--json` antes de qualquer um dos comandos a seguir para obter saída JSON estruturada: `init`, `validate`, `doctor`, `list`, `check`, `update`, `remove`.
</Note>

## Exemplos

<CodeGroup>

```bash Saída JSON
# Lista todos os apps em formato JSON
fp-appimage-updater --json list
```

```bash Config Personalizada
# Verifica atualizações usando um diretório de configuração personalizado
fp-appimage-updater -c ~/my-config check
```

```bash JSON e jq
# Atualiza um único app e extrai sua URL de download do JSON
fp-appimage-updater --json update whatpulse | jq '.[0].download_url'
```

</CodeGroup>
